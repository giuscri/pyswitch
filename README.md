# pyswitch

Easily switch between python2 and python3.

```
% python --version
Python 3.6.1
% sudo ./pyswitch -p2
✔️ python2
% python --version
Python 2.7.13
```

## Why?!
virtualenv's are boring. You should be able to easily mess
with your system.

## Dependencies
+ `python2-colorama`, for python2
+ `python-colorama`, for python3
